1.What is Virtual DOM?

Virtual DOM is the in-memory representation of Real DOM.React keeps a lightweight representation of the real DOM in the memory, and that is known as the virtual DOM. When the state of an object changes, virtual DOM changes only that object in the real DOM, rather than updating all the objects.

2.What is SPA?

SPA stands for Single Page Application. It is a very common way of programming websites these days. The idea is that the website loads all the HTML/JS the first time you visit. When you then navigate, the browser will only rerender the content without refreshing the website.

3.What is difference between Functional and Class component?

Functional Component

  1. It is used to create a dumb or stateless components.
  2. Functional component don't have any lifecycle methods they have hooks like UseState, UseEffect.
  3. Complete constructor structure is not required so they are easy to handle.
  4. They do not have state variables.

Class Component

  1. Class Components is used when we are doing data operations or server calls.
  2. Class Components have lifecycle methods.
  3. Complete constructor structure is required in class components.
  4. It contains state variables.

4.What does mean by state and its use in react?

The state is an instance of react component class can be defined as an object of a set of observable properties that control the behaviour of the component.

5.What is JSX and why do we use it instead of js?

JS is simply a scripting language adding functionality into a webpage but JSX is an additional to the JS syntax which is a mixture of both HTML and JavaScript.Both JS and JSX are interchangeable but JSX makes the code easier to understand for users.
